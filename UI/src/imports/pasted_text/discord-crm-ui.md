Create a complete premium web app UI for a Discord CRM and people intelligence platform.

This is not a normal dashboard. It should feel like a luxury, cinematic, high-budget digital product with custom art direction, rich motion, beautiful depth, and a polished premium identity. The entire experience should feel expensive, intentional, and emotionally designed — like it cost over $10,000 to design.

The product is a Discord CRM that collects and organizes people data from Discord:
- people directory
- user profiles
- knowledge bank
- message history
- media history
- admin settings
- analytics / activity views

The UI should feel like a hybrid of:
- luxury operating system
- futuristic creative studio product
- editorial fashion-tech site
- immersive glassmorphic memory archive

Avoid generic SaaS styling, boring admin panels, default dashboards, plain cards, or one-color flat interfaces. Everything should feel custom, spatial, elevated, and premium.

--------------------------------------------------
GLOBAL ART DIRECTION
--------------------------------------------------

Style:
- dark premium environment
- slightly grainy glassmorphism
- layered translucent surfaces
- subtle blur and frosted glass panels
- soft bloom and glowing edges
- rich shadows and atmospheric lighting
- cinematic contrast
- multi-tone palette, not monochrome
- visual depth everywhere
- premium typography and spacing
- motion-first thinking

Palette direction:
- deep navy
- charcoal black
- smoked violet
- muted steel blue
- cyan highlights
- soft silver
- subtle electric blue accents
- occasional premium warm accent for emphasis

Texture and finish:
- slight film grain
- subtle vignette
- soft noise overlays
- elegant reflection and refraction
- restrained glossy highlights
- not cartoonish, not over-neon, not too cyberpunk

Animation style:
- very smooth
- elegant spring transitions
- cinematic transitions
- soft floating motion
- layered parallax
- premium hover reactions
- morphing panels
- polished microinteractions
- motion that feels expensive, not gimmicky

--------------------------------------------------
INTRO / OPENING EXPERIENCE
--------------------------------------------------

Create an intro landing sequence before the main CRM appears.

The intro should last around 5 seconds and feel cinematic, smooth, mysterious, and premium.

Intro concept:
- dark atmospheric scene
- slight grainy glassmorphic overlays
- floating shards / glass panels / soft particles
- blurred profile silhouettes and ambient data fragments in the background
- subtle moving light sweeps
- elegant logo or product wordmark reveal
- a refined line like “Memory, identity, and relationships — mapped beautifully” or something similarly premium
- transition from abstract intro into the actual CRM interface

Animation behavior:
- slow fade-in of atmosphere
- soft drift of floating layers
- grain and blur resolving into clarity
- logo / title reveal
- elegant morph into the main app
- no cheesy loading spinner
- no cheap startup animation
- should feel like a luxury product intro

The intro should be skippable with a subtle “Enter” or “Skip” interaction.

--------------------------------------------------
APP STRUCTURE
--------------------------------------------------

Design the complete CRM with these main areas:

1. Intro / landing sequence
2. Main people directory / homepage
3. Individual person profile page
4. Knowledge bank view
5. Message and media history view
6. Admin settings panel
7. Optional analytics / system insights layer

--------------------------------------------------
1. MAIN PEOPLE DIRECTORY / HOMEPAGE
--------------------------------------------------

The homepage should be the emotional center of the app.

Core concept:
- a floating field / network / constellation of people
- circular profile pictures floating in space
- names beneath or near each profile
- subtle motion and spatial depth
- elegant distribution, not a rigid grid
- premium interactions and layered background motion

Include:
- floating profile portraits with names
- hover reveals summary details
- search bar integrated elegantly into the design
- filters for categories like:
  - active people
  - recently seen
  - most messages
  - most remembered
  - favorites
  - teams / groups / tags
- quick stats section:
  - total people
  - total remembered facts
  - recent activity
  - recent media
- ambient visual system:
  - subtle particles
  - connection lines
  - drifting glow
  - faint orbital structure
  - soft background grid or depth field

Interaction:
- hovering over a person adds glow, focus, and elevated motion
- clicking a person opens their detail page with a cinematic zoom/morph transition
- everything should feel immersive and elegant

--------------------------------------------------
2. PERSON PROFILE PAGE
--------------------------------------------------

When clicking a person, open a rich dedicated profile page.

This page should feel like a premium personal intelligence dossier.

Include:
- large hero profile section
- profile image / avatar
- full display name
- username / handle
- user ID
- status / activity indicator
- quick summary stats

Sections:
- remembered facts
- recent messages
- shared media
- projects / interests
- preferences
- timeline / activity history
- relationship or trust / memory signal
- tags / labels / categorization

Design behavior:
- information grouped beautifully, not dumped in plain tables
- layered panels with hierarchy
- expandable sections
- beautiful typography
- smooth transitions between subsections
- elegant chip/tag system
- subtle indicators for categories like identity, work, education, preferences, projects, contact, location

Media section:
- show images / videos / GIFs in a polished gallery
- use large previews, layered cards, or cinematic thumbnails
- premium lightbox / preview behavior

Timeline section:
- show recent observations over time
- blend messages, facts, and media in a timeline-like view
- clean and premium, not cluttered

--------------------------------------------------

Create a dedicated knowledge bank page that feels intelligent and structured.

Purpose:

- search

Use:
- elegant toggles
- segmented controls
- refined forms
- structured sections
- soft animations when changing tabs or settings
